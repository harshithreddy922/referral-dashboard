# Go Business Referral Dashboard

Frontend assessment project for a secure referral management dashboard.

## Features

- Login with the provided Go Business API credentials
- Cookie-based protected routes using `jwt_token`
- Referral dashboard with overview metrics, service summary, referral sharing, table search, date sorting, and client-side pagination
- Referral detail route loaded by referral id
- Public 404 page
